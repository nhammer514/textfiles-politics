Illuminati

New world Order

This site will be about a collection of conspiracies centered around around the Illuminati and the New World Order since these seem to be the basis of government and wide scale theories. Based on the files we have we will see how many and how often these two organizations are coorelated with. We will be using Relax NG schema to curate the files, since we are working with large file sets to help search through them for specific things we are looking for.
